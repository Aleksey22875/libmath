# libmath
Библиотека представляет собой набор математических и вспомогательных функций для выполнения базовых операций калькулятора: сложение, вычитание, умножение, деление, возведение в степень и факториал. Она не является самостоятельным приложением и не содержит точки входа — используется как зависимость в других проектах.

Сборка библиотеки отдельно не требуется, так как она хранится в удалённом репозитории GitHub и подключается напрямую через CMake как внешняя зависимость. При сборке основного проекта CMake автоматически подтягивает её исходники или подключаемый пакет и линковывает с исполняемым файлом.

Интеграция происходит на уровне CMake-конфигурации: библиотека объявляется как external dependency, после чего её функции становятся доступны в основном калькуляторе. Это позволяет использовать её как модульный слой вычислений без необходимости локальной сборки и ручного подключения.
